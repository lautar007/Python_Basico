# 🐍 Python Básico

## 📖 Introducción

**Python Básico** es un proyecto educativo diseñado para servir como material de estudio práctico sobre los fundamentos del lenguaje Python.

El repositorio presenta ejemplos comentados que permiten comprender de manera progresiva la sintaxis, las estructuras de control, el manejo de datos y los mecanismos básicos de modularización y persistencia de información.

Este proyecto está orientado al aprendizaje inicial de programación en Python, buscando fortalecer la lógica computacional y las buenas prácticas de organización del código.

## 🌐 Importancia de Python en la industria IT

Python se ha consolidado como uno de los lenguajes más relevantes dentro del ecosistema tecnológico moderno debido a su simplicidad, legibilidad y gran cantidad de bibliotecas disponibles.

Es ampliamente utilizado en desarrollo de software, análisis de datos, inteligencia artificial, automatización de procesos y ciberseguridad, entre otros campos. Su diseño favorece tanto el aprendizaje académico como el desarrollo profesional.

## 🛒 Mini proyecto de aplicación práctica

El proyecto incluye un ejemplo funcional de gestión de una lista de compras domésticas.

Este ejemplo integra múltiples conceptos fundamentales del lenguaje Python, utilizando el sistema de lectura y escritura de archivos como mecanismo de almacenamiento simple de información.

## ⚙️ Requisitos

- Python 3.x instalado en el sistema  
- Visual Studio Code (recomendado)  
- Extensión oficial de Python para Visual Studio Code

## 💻 Instalación y ejecución local

### 1. Clonar el repositorio

```bash
git clone URL_DEL_REPOSITORIO
```
### 2. Ingresar al directorio del proyecto

Abrir una terminal y ejecutar:

```bash
cd python-basico
```

### 3. Abrir el proyecto en Visual Studio Code

**Opción A — Desde la terminal**

```bash
code .
```

**Opción B — Manualmente desde VS Code**

1. Abrir Visual Studio Code  
2. Seleccionar **File → Open Folder**  
3. Buscar y seleccionar la carpeta del proyecto

> Recomendación: instalar la extensión oficial de Python en VS Code para linting, ejecución y debugging.

### 4. Verificar la instalación de Python

Comprobar la versión instalada:

```bash
python --version
```

Si tu sistema usa `python3` como comando:

```bash
python3 --version
```

### 5. Ejecutar el proyecto

Asegurarse de estar en el directorio del proyecto de ejemplo y ejecutar el archivo principal:

```bash
python main.py
```

Si el punto de entrada tiene otro nombre (por ejemplo `app.py` o `run.py`), reemplazar `main.py` por el nombre correspondiente.
